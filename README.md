# ag-grid-pagination

The repo tries to reproduce a grid pagination issue we see when the pagination size is flexible.

# To Run

Open two terminals

## Terminal 1

```
cd backend
npm run dev
```

## Terminal 2

```
cd frontend
npm run dev
```

To access, go to

```
http://localhost:3000
```

The backend repo includes a 500 record json of random data generated by https://generatedata.com/generator
